# 可视化旅游推荐系统接口设计文档

## 用户服务

1. **用户登录**:

- `method`: `POST`

- `url`: `/user/login`

- `Auth`: `N`

请求体:

```json
{
    "id": 0, // 登录阶段随便填写
    "username": "test_user1",
    "password": "123456",
    "gender": 0, // 默认值
    "city": ""   // ""
}
```

响应体:

```json
{
    "success": true,  // 响应是否成功
    "error": "", // 错误信息
    "data": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpZCI6NCwidXNlcm5hbWUiOiJ0ZXN0X3VzZXIxIiwiaXNzIjoieHp3IiwiZXhwIjoxNzY2MzcwMzc5LCJuYmYiOjE3NjU3NjQ1NzksImlhdCI6MTc2NTc2NTU3OX0.QIx-pyJA5zcYees2pCEMFIQM5U5G7aUHDxXKMr2zzKM"  // jwt token
}
```

> `jwt token` 中的负载信息为用户 `id` 以及用户名称, 这一个部分应该保存在前端

2. **用户注册:**

- `method`: `POST`

- `url`: `/user/register`

- `Auth`: `N`

请求体:

```json
{
    "id": 0,
    "username": "test_user1",
    "password": "123456",
    "gender": 1,          // 男 -> 0, 女 -> 1
    "city": "湖北省武汉市"  // 随便填
}

```

响应体:

```json
{
    "success": true,  // 响应是否成功
    "error": "",     // 错误信息
    "data": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpZCI6NCwidXNlcm5hbWUiOiJ0ZXN0X3VzZXIxIiwiaXNzIjoieHp3IiwiZXhwIjoxNzY2MzcwMzc5LCJuYmYiOjE3NjU3NjQ1NzksImlhdCI6MTc2NTc2NTU3OX0.QIx-pyJA5zcYees2pCEMFIQM5U5G7aUHDxXKMr2zzKM"  // jwt token
}
```

3. **用户信息查询:**

- **`method`: `POST`**
- **`url`: `/userInfo/user`**
- **`Auth`: `Y`**

**请求体: 无**

**响应体:**

```json 
{
    "success": true,
    "error": "",
    "data": {
        "id": 4,
        "username": "test_user1",
        "password": "123456",
        "gender": 1,  // 性别 0 -> 男  |  1 -> 女生
        "city": "湖北省武汉市"
    }
}
```

## 点赞服务

1. **点赞:**

- **`method`: `GET`**
- **`url`: `/like/like/:siteIndex` (其中 `siteIndex` 为动态参数, 景点索引)**

- **`Auth`: `Y`**

请求体: 无

**响应体:**

```json
{
    "success": true, // 表示点赞是否是否成功
    "error": "",
    "data": ""
}
```

2. **取消点赞:**

- **`method`: `GET`**
- **`url`: `/like/cancel/:siteIndex` (其中 `siteIndex` 为动态参数, 景点索引)**
- **`Auth`: `Y`**

请求体: 无

**响应体:**

```json
{
    "success": true,
    "error": "",
    "data": ""
}
```

3. **查询某一个景点是否被当前用户点赞:**

- **`method`: `GET`**
- **`url`: `/like/isLike/:siteIndex` (其中 `siteIndex` 为动态参数, 景点索引)**
- **`Auth`: `Y`**

请求体: 无

**响应体:**

```json
{
    "success": true,
    "error": "",
    "data": true  // 是否点赞
}
```

4. **查询当前用户的点赞景点列表:**

- **`method`: `GET`**
- **`url`: `/like/siteList`**
- **`Auth`: `Y`**

请求体: 无

**响应体:**

```json
{
    "success": true,
    "error": "",
    "data": {
        "data": [  // 注意这里是景点索引的列表
            1,
            4,
            5
        ],
        "total": 3
    }
}
```

- **`url`: `/like/userList/:siteIndex`**
- **`Auth`: `Y`**

请求体: 无

**响应体:**

```json
{
    "success": true,
    "error": "",
    "data": { 
        "data": [ // 用户 id 列表
            3,
            4
        ],
        "total": 2
    }
}
```

## 浏览记录服务

1. **记录浏览记录:**

- **`method`: `GET`**
- **`url`: `/view/view/:siteIndex`**
- **`Auth`: `Y`**

请求体: 无

**响应体:**

```json
{
    "success": true,
    "error": "",
    "data": false   // 浏览记录是否添加成功, 如果请求成功但是 data 为 false 表示浏览过了 ; data 为 true 表示浏览记录添加成功
}
```

> **注意这里如果 `success` 字段为 `true` 但是 `data` 为 `false` 表示用于已经浏览过了**

2. **查询当前用于浏览过的所有景点列表:**

- **`url`: `/view/siteList`**
- **`Auth`: `Y`**

**响应体:**

```json

    "error": "",
    "data": {
        "data": [ // 景点 siteIndex 列表
            1,
            2,
            3
        ],
        "total": 3
    }
}
```
