# 可视化旅游推荐系统接口设计文档

## 用户服务

1. **用户登录**:

- `method`: `POST`

- `url`: `/user/login`

请求体:

```json
{
    "id": 0, // 登录阶段随便填写
    "username": "test_user1",
    "password": "123456",
    "gender": 0, // 默认值
    "city": ""   // ""
}
```

响应体:

```json
{
    "success": true,  // 响应是否成功
    "error": "", // 错误信息
    "data": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpZCI6NCwidXNlcm5hbWUiOiJ0ZXN0X3VzZXIxIiwiaXNzIjoieHp3IiwiZXhwIjoxNzY2MzcwMzc5LCJuYmYiOjE3NjU3NjQ1NzksImlhdCI6MTc2NTc2NTU3OX0.QIx-pyJA5zcYees2pCEMFIQM5U5G7aUHDxXKMr2zzKM"  // jwt token
}
```

> `jwt token` 中的负载信息为用户 `id` 以及用户名称, 这一个部分应该保存在前端

2. 用户注册:

- `method`: `POST`

- `url`: `/user/register`

请求体:

```json
{
    "id": 0,
    "username": "test_user1",
    "password": "123456",
    "gender": 1,          // 男 -> 0, 女 -> 1
    "city": "湖北省武汉市"  // 随便填
}

```

响应体:

```json
{
    "success": true,  // 响应是否成功
    "error": "",     // 错误信息
    "data": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpZCI6NCwidXNlcm5hbWUiOiJ0ZXN0X3VzZXIxIiwiaXNzIjoieHp3IiwiZXhwIjoxNzY2MzcwMzc5LCJuYmYiOjE3NjU3NjQ1NzksImlhdCI6MTc2NTc2NTU3OX0.QIx-pyJA5zcYees2pCEMFIQM5U5G7aUHDxXKMr2zzKM"  // jwt token
}
```
